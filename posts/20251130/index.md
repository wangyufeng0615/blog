这个周末我重构了我的博客。对我的博客来说，重构意味着只保留历史上的文章，但更新软件框架。比如，最早我独立部署博客的时候，用的是hexo。2023年8月，我换成了hugo。而这一次重构，也许是我的最后一次对博客重构，因为这次没有依赖任何框架。

从2023年初开始用ChatGPT 3.5开始，我就想要着手重写自己的博客，因为这是我一直以来的心愿，也就是纯靠我自己（当然有了AI以后会容易很多）写我博客的前端页面，正好借此练手一下前端。但因为我那时候确实对前端不太熟悉，AI Coding的能力又不太好，这个事情就一直耽搁。

大概是到了今年年初，那时候好像是用上了Cursor + Claude sonnet 3.5，我又尝试了重写博客这件事，但那时候前端的样式总是让我不够满意，总觉得自己想要的效果AI搞不出来，或者搞出来的页面我还得做各种微调，就又搁置了这个事情。周末我整理自己的老代码的时候，突然想起了博客这件事，我就用上周刚出的Claude Opus 4.5做了一下尝试：果然不一样了，大概对话了4-5个回合，一个完整的、效果令我满意的「纯react + markdown渲染库 + github actions部署」的博客就写好了，也就是当前博客的样式。

为什么说这也许是我最后一次重构博客，因为如果我不靠任何前端「更高级的框架」，比如hexo、hugo、wordpress这种框架，而只是靠最基本的React甚至HTML+CSS+JS，配合AI，就能写出我想要的页面，那确实好像再也没有「更换框架」的空间了。我想，React足够主流，也许不至于回退到纯HTML+CSS+JS的程度。

这件事也引发了我的一个思考，或者说一个判断：今后的软件工程里，主流的框架会变得越来越主流，不主流的框架会越来越被淘汰，因为足够主流的框架，网络上的资料足够多，代码量足够多，相关的讨论也足够多，AI掌握的越好；而AI掌握的越好，人们就更倾向于去使用更主流的框架解决问题。

框架的一个作用是提供开箱即用的现成功能。比如我的这个博客，需要将markdown在发布前转为静态HTML，这个功能如果放在以前没有AI的时候，我最好是用一个现成的框架，比如Hexo，要不然我还得自己开发，而现在这变得非常容易。或者，如果我还是坚持使用Hexo，但AI对这个框架的理解比较少，我强行让AI使用Hexo的话，可能会出现各种各样的问题，还不如让AI基于它最了解的技术栈去现场构建所需要的功能，例如markdown渲染。

所以这就是我对未来软件工程领域的一个判断：少数主流的框架会变得越来越主流，成为AI所最为熟悉和掌握的底层能力的一部分，而不够主流的框架将会逐渐消亡，因为这些框架所能带来的收益已经要小于AI不够熟悉这些框架所带来的成本。毕竟，当有Opus 4.5这样能力程度的AI存在的情况下，「生成代码」这件事的成本降到了一个以往也许任何一个软件工程师都难以想象的水平。

拓展开来畅想的话，我想，不只是非主流的框架会面临这样的局面，而是一切软件都将会面临这样的局面。Hexo这样的框架为何存在，是因为这个世界上存在对Hexo这样框架的「需求」。而在AI的影响下，很多「对软件的需求」都将不复存在。比如说，现在我用AI开发一个可视化Web系统，将会变得非常容易（至少从些代码时间成本的角度考虑），但是，原本需要这样一个可视化页面的用户，他自己也可以使用AI去获取他想要的可视化界面，他为什么还要依赖另外一个程序员给他定制好的软件呢？

现在人们可能会说，是因为「用户」编码的技能还不够好，自己还不能驾驭AI，但如下这个趋势发生的非常快，也就是AI对人类软件工程师的依赖程度正在快速下降。ChatGPT 3.5时期，他写出的代码我基本上得逐行Review，在Cursor时期，我得把他当成不太利索的实习生，在claude code + sonnet 4时期，他是我的初级工程师同事，而现在的Opus 4.5，我认为在很多的情况下，他已经可以当我的mentor了。

对于普通客户来说也是一样的，现在虽然非工程师用户，还很难使用claude code或者豆包去生成自己需要的软件，但我相信这是必然的趋势，也就是今后任何稍有电脑操作基础的用户，都有能力去生成自己需要的软件。除非这个用户完全没有任何自己定制化的需求，那就用现成的App就足够了，比如短视频App的用户。

我的这个博客，我想我会一直维护下去的，我甚至希望这个域名能比我的生命的长度更加久远，我希望我的后代，如果有的话，能记得一直帮我续费注册这个域名，直到那个我们谁也无法预测的遥远未来，我相信这个未来所遥远的程度是，那时候的人们看待今天的人类社会，就像是大人在看待一个刚刚学会了一些简单的词汇，正在胡言乱语四处乱爬的婴儿。在这样一个Web3的时代，一个AI的时代，我还能有这么一个小小的赛博空间，用手敲文字的方式去记录我的一些感受，也许在一个遥远的未来时代，我这样的人会被称为web2.0时代的老古董，被AI考古学家记录在某个冷存储介质上。
